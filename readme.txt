Create custom questions, deploy via shortcodes, manage responses, mark favorites or ban.

== Description ==

This plugin empowers users to craft unique questions, seamlessly integrating them into custom pages through shortcodes. Users can not only receive responses but also exert control by marking answers as favorites for future reference or banning those that are undesirable. This comprehensive tool enhances user engagement and flexibility within the WordPress platform.

== Installation ==

1. Upload the entire `smart-answer` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Usage ==

1. After activation, go to the plugin settings to create custom questions.
2. Copy the shortcode provided for your question.
3. Paste the shortcode into any post or page where you want the question to appear.


== Frequently Asked Questions ==

= How do I create a custom question? =
Navigate to the plugin settings, locate the "Create Question" option, and follow the prompts to design your personalized question.

= How can I use the question on a custom page? =
After creating a question, you'll receive a shortcode. Simply copy and paste this shortcode into the desired custom page or post.

= Can I manage the answers I receive? =
Yes, within the plugin interface, you can view, favorite, or ban specific answers associated with your custom question.

= What happens when I mark an answer as a favorite? =
Favoriting an answer allows you to easily reference and retrieve it later for quick access. 

= How does the banning feature work? =
Banning an answer ensures it won't be displayed, providing control over content quality.

= Is there a limit to the number of questions I can create? =
The plugin supports the creation of multiple custom questions, allowing for diverse and personalized content.

= How do I uninstall the plugin if needed? =
Navigate to the WordPress dashboard, go to Plugins, find the plugin "Smart Answer", and click "Deactivate" and then "Delete".
